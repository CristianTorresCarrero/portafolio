Sistema Ventas Frozen Flavuors

El Sistema de Ventas "Frozen Flavuors" es una aplicación de escritorio desarrollada en Java con MySQL para la gestión eficiente de ventas en una heladería. Este sistema permite administrar productos, empleados, clientes e inventario, además de generar reportes gráficos y descargar recibos de venta en formato PDF. Su objetivo principal es optimizar el control de ventas y mejorar la administración del negocio.

Ver Proyecto

HoraSocial

HoraSocial es una aplicación móvil desarrollada en Kotlin con Firebase, diseñada para la gestión eficiente de los horarios laborales en una empresa. Su objetivo principal es permitir que la administradora agregue, modifique y asigne horarios de trabajo, mientras que los empleados pueden visualizarlos en tiempo real desde sus dispositivos móviles.
